# ALGORITHM-SCOPE.md

한국 정보 교육과정 기준으로 파이썬으로 다루는 알고리즘을 정규화한 문서.
문제 출제 범위/난이도 판단의 기준으로 사용.

## 1. 교육과정 층위별 알고리즘 취급

파이썬으로 실제 구현까지 가는 본체는 정보과학(심화)이다.

| 과목 | 위치 | 알고리즘 깊이 | 언어 |
|------|------|--------------|------|
| 중학교 정보 | 중1~3 | 순차/선택/반복, 순서도, 탐색/정렬 개념 | 블록 위주 |
| 고1 정보(일반) | 공통/일반선택 | 알고리즘 설계, 시간복잡도(효율성) 개념 | Python/C |
| 인공지능 기초 | 진로선택 | 탐색 기반 문제해결(최상우선, A*) | Python |
| 정보과학(심화) | 진로선택 | 자료구조 + 정렬/탐색/그래프/DP 전부 구현 | Python/C |

- 2015 개정: 정보과학이 알고리즘 본체.
- 2022 개정: 정보과학 유지, "데이터 구조 + 알고리즘" 2대 영역으로 재편.
  동적계획법(메모이제이션), 인접리스트 기반 그래프/트리를 성취기준에 명시.

## 2. 정규화된 알고리즘 카탈로그

정보과학 자료처리 영역([12정과02-01~05]) + 관계기반 알고리즘 영역 기준.

### A. 자료구조 (토대)
- 선형: 배열/리스트, 스택, 큐 (덱, 연결리스트)
- 비선형: 트리(이진트리), 그래프(인접행렬 / 인접리스트)
- 부가: 힙(우선순위 큐), 해시테이블

### B. 정렬 (Sorting)
- 기초 O(n^2): 선택 정렬, 버블 정렬, 삽입 정렬
- 고급 O(n log n): 병합 정렬, 퀵 정렬, 힙 정렬
- 초점: 구현 + 효율성 비교/분석(성취기준 명시)

### C. 탐색 (Searching)
- 선형 자료 탐색: 순차 탐색, 이진 탐색
- 그래프/트리 탐색: DFS(깊이 우선), BFS(너비 우선)
- 지능형 탐색(AI 기초): 최상 우선 탐색, A* 탐색

### D. 알고리즘 설계 기법 (Paradigm)
- 완전탐색(brute force) / 백트래킹(분기한정)
- 재귀(recursion), 분할정복(병합/퀵/이진탐색이 여기 속함)
- 탐욕법(Greedy)
- 동적계획법(DP): 하향식(메모이제이션) / 상향식(테이블), 부분문제 재귀관계 정의

### E. 알고리즘 분석
- 시간/공간 복잡도, 빅오 표기, 알고리즘 성능 비교

### F. 그래프 심화 (정규 성취기준 밖, 교과서 심화/경시 참고)
- 최단경로: 다익스트라
- 최소신장트리(MST): 크루스칼, 프림

## 4. 출제 관점 요약

- 뼈대: 자료구조 -> 정렬 -> 순차/이진탐색 -> DFS/BFS
- 확장: 설계기법(재귀/분할정복/그리디/DP/백트래킹) + 복잡도 분석
- F(다익스트라/MST)는 정규 밖이나 심화/경시에서 자주 등장.
