# 기계학습 이론 요약 (학생 제출본)

학생이 수업에서 배운 내용을 정리해 온 요약. 일부 누락 가능성 있음(원문 그대로 보존).

## 기계학습을 이루는 3가지 학습법

### 1. 지도학습
- 특징: 정답지가 있는 학습
- 키워드: 입력 데이터 + 정답(레이블)
- 목적: 어디에 속하는지 예측
- 장점: 레이블이라는 정답이 있어 학습 모델을 명확하게 평가, 평가 결과에 따라 성능 개선이 쉬움

용어:
- 독립변수: 예측에 사용되는 값
- 종속변수/레이블: 우리가 맞히고 싶은 진짜 정답

분류 알고리즘:
- 분류1: k-NN(최근접 이웃) - 거리 측정 / 다수결 투표 / 하이퍼파라미터
- 분류2: 의사결정트리 - 스무고개처럼 예/아니오 질문을 던져 정답을 좁혀가는 직관적 방식
- 분류3: 앙상블과 랜덤 포레스트 - 여러 전문가의 의견을 모아 예측
- 분류4: 로지스틱 회귀 - 예측이 아니라 분류하는 방식, 선을 그려 구역을 나눔, 로지스틱 함수 1/(1+e^(-x))

회귀:
- 종속변수로 독립변수를 예측하는 방법
- 예) 선형회귀

### 2. 비지도학습
- 특징: 정답 없이 패턴을 찾는 학습
- 키워드: 데이터의 숨은 구조 파악
- 목적: 군집으로 그룹화할 때
- 장점: 명확한 정답을 제공하진 않지만, 학습을 통해 데이터에 숨겨진 패턴을 발견 가능

군집 분석 알고리즘(k-Means): 비슷한 특성을 가진 것들을 k개의 군집으로 나누는 방식
1. 초기화
2. 배정
3. 재설정
4. 종료

군집화 과정:
1. 과일 분석하기
2. 수직선에 나타내기
3. 그래프로 표시하여 그룹 나누기

### 3. 강화학습
- 특징: 보상을 통해 행동을 수정하는 학습
- 키워드: 환경, 행동, 보상 최대화

## 기계학습 문제 해결 맵 (알고리즘 가이드)

1. 데이터에 정답이 있나요?
   - yes: 지도학습
   - no: 비지도학습
2. (지도학습일 때) 예측하려는 값이 숫자인가요, 범주인가요?
   - 숫자: 회귀
   - 범주: 분류
