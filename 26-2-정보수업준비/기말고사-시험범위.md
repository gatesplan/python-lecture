## 클래스와 객체 개념 설명

- 클래스 = 붕어빵 틀, 객체 = 실제 붕어빵: 틀(클래스)은 하나지만 객체는 여러 개 생성 가능
- 클래스 안에는 속성(attribute)과 메서드(method, 동작 함수)가 포함됨
- `__init__`: 객체 생성 시 초기 속성값을 설정하는 메서드, 거의 모든 클래스에 포함
- `self`: 클래스 자기 자신을 가리키며, 클래스 내부 속성에 접근할 때 사용 (`self.name`, `self.hp` 등)

## 게임 캐릭터 클래스 예제

- `Character` 클래스: `__init__`으로 `name`, `hp` 속성 초기화
- `hero = Character("파이썬 기사", 100)` -> `name="파이썬 기사"`, `hp=100`인 객체 생성
- `monster = Character("슬라임", 30)` -> 동일 클래스에서 다른 객체 생성
- `hero.say_hello()`: 히어로의 `say_hello` 메서드 실행, `self.name` 출력
- `monster.take_damage(10)`: 몬스터의 HP를 10 감소

## 자동차 클래스 예제 (연습 문제)

- `Car` 클래스: 속성으로 `color`, `speed`(기본값 0) 설정
- 메서드: `speed_up(amount)` -> 속도 증가, `stop()` -> 속도 0으로 초기화
- `my_car = Car("red")` 생성 후 `my_car.speed_up(50)`, `my_car.stop()` 실행

## 함수 복습 (클래스 이해를 위한 사전 정리)

- 두 숫자를 입력받아 합계를 반환하는 함수 `jw(a, b)` 예시로 함수 구조 복습
- 함수의 네 가지 유형: 넘겨주고 돌려받음 / 넘겨주고 돌려받지 않음 / 넘겨주지 않고 돌려받음 / 둘 다 없음
- 클래스 내부의 메서드도 함수와 동일한 방식으로 동작하며, 본문에서 호출해 사용
