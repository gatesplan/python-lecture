# 26-2 정보수업 - 데이터의 압축과 암호화 (260516)

## 1. 데이터 압축

### 정의
원본 데이터를 작은 용량으로 압축시키는 것.

### 압축의 종류
- **무손실 압축**: 손실 없음. ZIP, GIF, PNG 등 (예: 은행 기록처럼 원본 보존 필요)
- **손실 압축**: 손실 있음. JPEG, MPEG, MP3 등

### 데이터 압축의 필요성
- 전송할 파일을 압축함으로써 데이터 전송 시간을 단축할 수 있음
- 전송할 때 쓰이는 에너지를 줄여 효율적임
- 압축된 데이터를 풀면 본래 형식으로 풀어내므로 안전하게 운용 가능

### 데이터 단위
| 종류 | 크기 |
|---|---|
| 숫자 | 1 byte |
| 영문 | 1 byte |
| 한글 | 2 byte |

| 단위 | 크기 |
|---|---|
| 1 KB | 1024 byte |
| 1 MB | 1024 KB |
| 1 GB | 1024 MB |
| 1 TB | 1024 GB |

## 2. 무손실 압축 기법

### (1) 런 길이 부호화 (Run-Length Encoding)
같은 값이 연속되는 길이로 표현하여 압축.

예시:
```
AAADDB0000  ->  A3D2B1O4
(10 byte)       (8 byte)
```

### (2) 허프만 부호화 (Huffman Coding)
절차:
1. 문자의 빈도수 정리 및 내림차순 정렬
2. 빈도 가장 작은 둘로 묶기 (작은 쪽부터)
3. 이진트리 붙이기 (왼쪽: 0, 오른쪽: 1)
4. 데이터 변환하기

예시: `AAADDB0000`
- 빈도: A:3, B:1, D:2, 0:4
- 내림차순: 0:4, A:3, D:2, B:1
- 트리 구성 후 코드 부여 (예: A->00, D->010, B->011, 0->1 형태)
- 부호화 결과: 00000010001111 (예시)

## 3. 카이사르 암호화

글자를 일정 칸 수만큼 밀어서 치환하는 암호화.

예시:
- 원문: `abcd`
- 코드값(이동량): 4
- 결과: `efgh`

## 4. 이미지 파일 정보 출력 (Python)

PIL 라이브러리와 os 모듈을 이용하여 이미지의 확장자, 해상도, 파일 크기를 출력.

```python
# 도구 가져오기
import os
from PIL import Image

# 이미지 불러오기
im = Image.open('충남과학고.bmp')

# 정보 출력하기
print('이미지 확장자', im.format)
print('이미지 해상도', im.size)
print('파일 크기', os.path.getsize('충남과학고.bmp'), 'Bytes')
```

### 코드 구성
| 단계 | 코드 | 설명 |
|---|---|---|
| 도구 가져오기 | `import os`, `from PIL import Image` | 파일 크기 조회용 os, 이미지 처리용 PIL |
| 이미지 불러오기 | `Image.open(파일명)` | 이미지 파일을 메모리로 로드 |
| 정보 출력 | `im.format`, `im.size`, `os.path.getsize()` | 확장자/해상도/크기 |
