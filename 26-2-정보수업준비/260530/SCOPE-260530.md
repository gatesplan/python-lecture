## 1. 이론 필기 (note1.jpg)

### 네트워크
- IP주소: 컴퓨터를 구분할 수 있는 고유 숫자
  - IPv4 = 8비트씩 4자리 = 총 2^32. 주소 부족으로 IPv6의 128비트로 전환 중
- 도메인 이름 (연락처 이름에 비유)
- DNS 서버: 도메인을 입력하면 알아서 IP로 바꿔줌
  - 예) www.naver.com <-> 10.113.12 (도메인 <-> IP)
  - www -> 프로토콜, naver -> 기관
  - 도메인 구조 예) yahoo.co.kr = 기관 / 성격 / 국가  (기관:co, 공공:go)
- 네트워크 종류: 유선 vs 무선
  - 유선: 속도가 빠르고 신뢰할 수 있지만, 이동이 어려우며 선이 복잡함
  - 무선: 물리적 연결 없이 이동하기 편리하지만, 보안에 취약하며 거리 제한 및 신호 간섭 등이 존재함

### 로컬 환경 공유 (예: 공유 폴더, 에어드랍 등)
  1. 최소 공유
  2. 사전 검사
  3. 정보 보호
  4. 권한 설정

### 사물인터넷 (IoT)
- IoT = Internet of Things
- 구성 요소 D / N / P:
  - D (센서): 예) 마이크로컨트롤러 등
  - N (네트워크): NFC, RFID, Beacon, ZigBee, WiFi 등
  - P (플랫폼): 데이터를 수집하여 유용한 정보 생산
