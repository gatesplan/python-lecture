# 5-1 디지털기술과 사회변화

## 페이지 1: 표지
- 제목: 5-1 디지털기술과 사회변화

## 페이지 2: 유비쿼터스 - 끊임없이 연결된 디지털 생태계
현대 사회의 디지털 기술은 언제 어디서나 접속이 가능한 유비쿼터스(Ubiquitous) 환경을 기반으로 합니다.

핵심 4가지 영역:
1. 스마트 시티
2. 스마트 홈
3. 웨어러블
4. 글로벌 네트워크

내장된 센서가 주변 환경을 인지하고 사회와 실시간으로 통신하여 정보를 제공합니다.

## 페이지 3: 디지털 엔진의 4가지 핵심 기둥
데이터는 어떻게 우리의 삶을 바꾸는가? 4가지 핵심 기술이 하나의 시스템으로 연결되어 작동합니다.

- [수집/감각] 사물 인터넷 (IoT): 현실 세계의 데이터를 실시간으로 모으는 감각 기관.
- [저장/기억] 클라우드 (Cloud): 물리적 한계 없이 방대한 데이터를 저장하고 공유하는 가상 공간.
- [분석/두뇌] 빅데이터 (Big Data): 쌓인 정보 속에서 가치 있는 패턴과 원인을 찾아내는 지식의 도서관.
- [실행/결정] 인공지능 (AI): 데이터를 학습하여 스스로 판단하고 최적의 결과를 실행하는 의사결정자.

## 페이지 4: 물리적 한계를 넘어서 - 수집과 저장
사물 인터넷과 클라우드는 데이터를 모으고, 언제 어디서든 접근할 수 있게 만듭니다.

- 사물 인터넷 (IoT): 온도, 습도를 스스로 감지하고 조절하여 의약품 운송의 안전성을 극대화합니다.
- 클라우드 (Cloud): USB와 같은 이동형 저장장치 없이도, 전 세계 어디서든 적은 비용으로 데이터를 공유합니다.

## 페이지 5: 데이터에서 지능으로 - 분석과 실행
빅데이터와 인공지능은 복잡한 문제의 원인을 찾고 스스로 해결책을 실행합니다.

- 빅데이터 (Big Data): 정확한 진단. 특정 질환 환자들의 행동 데이터를 분석하여 숨겨진 병의 원인을 찾아냅니다.
- 인공지능 (AI): 산불 발생 지역 탐지. 딥러닝 기술을 통해 기계가 인간을 대신해 위험한 산불 발생 지역을 스스로 탐지합니다.

## 페이지 6: 통합 사례 - 디지털 트윈과 스마트 안전 축사
현실의 문제를 해결하기 위해 디지털 공간에 완벽한 쌍둥이를 만듭니다.

- 센서 데이터 수집: 바이오센서와 24시간 카메라가 생육 상태 전송.
- 가상 시뮬레이션: 클라우드와 빅데이터로 최적의 환경(온도/질병 예측) 분석.
- AI 자율 제어: 현실 축사의 환경을 자동으로 조절하여 생산성 극대화 및 질병 예방.

## 페이지 7: 다가오는 미래 - 일상 속 첨단 기술
첨단화, 지능화된 디지털 기술은 우리의 몸, 이동 수단, 그리고 환경을 완전히 새롭게 정의합니다.

- 메타버스 (Metaverse): 3차원 가상 공간의 확장
- 웨어러블 (Wearables): 생체 신호와 건강 예측
- 자율주행 (Autonomous Driving): 조작 없는 이동의 자유
- 투명 모니터 (Transparent Displays): 공간의 정보화

## 페이지 8: 미래 기술 1 - 무한한 경험과 생체 데이터
가상과 현실의 경계가 무너지고, 기술은 우리의 신체와 하나가 됩니다.

- [메타버스 (Metaverse)] 실세계와 같은 사회, 문화 활동이 이루어지며, 가상 공간 내 상호작용으로 새로운 가치를 창출합니다.
- [웨어러블 (Wearables)] 반지, 안경 형태의 기기가 심박수, 뇌파 등 생체 신호를 실시간으로 감지하여 건강 이상을 미리 예측합니다.

## 페이지 9: 미래 기술 2 - 노동의 진화와 공간의 혁신
기계가 스스로 일하고, 모든 유리창이 정보를 제공하는 디스플레이가 됩니다.

- [자율주행 (Autonomous Driving)] 외부 환경을 스스로 인지하고 주행합니다. 농업 인력 부족 문제를 완화하고 작업 효율을 높입니다.
- [투명 모니터 (Transparent Monitors)] 유리창이 TV와 광고판으로 변신하여, 이동 중에도 쉽게 각종 업무를 처리하고 정보를 확인합니다.

## 페이지 10: 직업 패러다임의 전환 - 아날로그에서 디지털로
기존의 직업들은 기술과 결합하여 그 수행 방식이 근본적으로 변화하고 있습니다.

| 직업 (Profession) | 아날로그 방식 (Analog Past) | 디지털 전환 (Digital Future) |
| --- | --- | --- |
| 간호사 (Nurse) | 사람이 직접 체중/신장을 측정 및 기록 | 신체 계측 자동화 시스템과 의료 데이터 연동 |
| 심판 (Umpire) | 인간의 육안에 의존한 판정 | 초고속 카메라 및 AI VAR 인식 결과 기반 판정 |
| 만화 작가 (Artist) | 종이와 펜을 이용한 물리적 작화 | 태블릿과 생성형 AI를 활용한 자유로운 창작 |

## 페이지 11: 새로운 진로의 개척 - 디지털 시대의 유망 직업
수많은 분야에서 디지털 기술이 융합되며, 과거에는 없던 완전히 새로운 직업들이 탄생하고 있습니다.

디지털 융합 (Digital Convergence)
- 인공지능 전문가: AI 프로그램 개발 및 학습 설계
- 스마트 안전 관리사: VR, AR, 드론을 활용한 건설/산업 현장 사고 예방
- 미래 자동차 전문가: 전기차, 자율주행, 커넥티드카 연구
- 환경 빅데이터 전문가: 환경 데이터를 구축하여 기후 위기 의사결정 지원
- VR/AR 콘텐츠 기획자: 교육, 군사, 관광, 예술 등 가상현실 경험 설계

## 페이지 12: 인간 vs 기계 - 새로운 공존의 시대
기계가 인간의 노동을 대체하는 시대, 우리는 어떤 능력을 키워야 할까요?

- [기계의 영역 (Machine's Domain)]
  - 방대한 데이터 수집
  - 24시간 끊임없는 연산
  - 위험 지역 탐지
  - 반복적이고 정밀한 작업
- [인간의 영역 (Human's Domain)]
  - 맥락 이해
  - 창의적 문제 해결
  - 윤리적 판단
  - 타인과의 공감

기술을 도구로 활용하여 나만의 강점을 만드는 5가지 핵심 역량이 필요합니다.

## 페이지 13: 미래를 설계하는 힘 1 - '생각'의 역량
디지털 생태계의 복잡한 정보를 해석하고 새로운 가치를 창출하는 3가지 인지적 역량.

- 컴퓨팅 사고 (Computational Thinking): 일상과 학문의 문제를 컴퓨터의 데이터 처리 방식처럼 논리적으로 분석하고 빠르고 정확하게 해결하는 능력.
- 지식 정보 처리 (Information Processing): 쏟아지는 지식과 정보를 비판적으로 탐구하고, AI 기술을 올바르게 이해하여 활용하는 능력.
- 창의적 사고 (Creative Thinking): 폭넓은 기초 지식을 바탕으로 다양한 기술과 경험을 융합하여 세상에 없던 새로운 것을 만들어 내는 능력.

## 페이지 14: 미래를 설계하는 힘 2 - '연결'의 역량
기계가 할 수 없는, 사람과 사람을 잇고 지속 가능한 내일을 만드는 2가지 사회적 역량.

- 협력적 소통 (Collaborative Communication): 타인의 관점을 존중하고 내 생각을 효과적으로 표현하여, 공동의 목적을 위해 협력하는 인간 고유의 소통 능력.
- 공동체 역량 (Community Competence): 내가 속한 세계를 이해하고, 지속 가능한 인류 발전을 위해 책임감 있게 디지털 기술을 사용하는 윤리적 태도.

## 페이지 15: 당신의 디지털 청사진을 그려라
미래는 수동적으로 맞이하는 것이 아니라, 여러분의 역량으로 직접 설계하는 것입니다.

디지털 문해력과 창의성을 무기로, 다가오는 디지털 시대의 진정한 설계자가 되십시오.
