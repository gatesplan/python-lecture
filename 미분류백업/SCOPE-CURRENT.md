# 현재 학습 범위 (출제 범위 필터)

## 출력

- `print()`: 기본 사용
- `sep`, `end` 파라미터 사용 가능
- f-string 금지
- 정수, 실수, 텍스트 변수 출력

## 자료형

- `int`, `float`, `str`, `bool`
- `list`, `dict`, `tuple`
- `type()` 함수 사용 가능

## 리스트

- 인덱스 접근
- 기본 메서드: `append`, `clear`, `remove` 등 기본 입출력 관련
- 전체 메서드를 다 배운 상태는 아님

## 딕셔너리

- `dict[key]` 직접 참조
- `get()` 메서드
- 선생님이 비전문가라 학생들이 혼란 있음

## 튜플

- 기본 사용

## 제어문

- `if` / `elif` / `else`
- `for` + `range()` (start, stop, step 모두 가능)
- `while`
- `break`, `continue`: 배웠을 가능성 높으나 불확실

## 함수

- `def`로 함수 정의
- 위치 매개변수 (0개, 1개, 여러 개)
- `return` 으로 값 반환
- 반환값 사용: 변수에 대입, `print()` 인자로 직접 전달
- 함수 내부 지역변수 사용 가능
- 기본값/키워드 인자, `*args`/`**kwargs`는 아직 미학습

## 문자열

- 정식 수업 없음, 중구난방으로 사용 중
- 체계적 학습 상태 아님

## 금지/미학습

- 클래스
- f-string
- `ord()`, `chr()`
- 고급 내장함수 (`map`, `filter`, `enumerate`, `zip` 등)
- 리스트 컴프리헨션
- 파일 입출력
- 모듈/import

## 비고

- 학교 담당 교사가 비전문가로 추정, 커리큘럼이 체계적이지 않음
- 학생들의 실제 이해도와 수업 범위에 괴리 있을 수 있음
