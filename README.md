# Python Lecture Materials

파이썬 강의 자료 아카이브

## 구성 / Contents

모든 자료는 `legacy/` 폴더에 정리되어 있습니다.

### 강의 노트북 / Lecture Notebooks (`legacy/notebooks/`)

| 폴더 | 내용 | 주제 |
|---|---|---|
| `01-basic/` | Python 기초 | 리스트, 입출력, 조건문, 반복문, 튜플, 함수 |
| `02-intermediate/` | 자료구조/알고리즘 | 스택, DFS, 미로탐색, 하노이탑, 트리순회, 시간복잡도, 백트래킹 |
| `03-design-patterns/` | OOP/디자인패턴 | 클래스, Composite, Factory, Builder, Strategy, Observer |
| `04-machine-learning/` | 머신러닝 | 선형/다항회귀, SVM, KNN, 의사결정트리, 클러스터링, 뉴럴네트워크 |
| `05-rne-statistics/` | R&E 통계분석 | 단변량, 이변량, 다변량 분석 |
